=====================
Icons Usage Helper
=====================

**You can use the icons provided by this extension in your Fluid frontend or backend templates using the core:icon ViewHelper.**

#. Select the 'Pages icons' module on the left.
#. Select the 'Icons usage helper' submodule on top.
#. Select an icon.
#. Click on the 'Copy' button to copy the corresponding core:icon ViewHelper.
