==================
Search For Icons
==================

#. Select the 'Pages icons' module on the left.
#. Choose either 'Page icon changer' or 'Icons usage helper' submodule.
#. Use the search box to find a specific icon. You can just type in a color, a word or several words.
#. Press enter on keybord or click on the magnifier icon button.
