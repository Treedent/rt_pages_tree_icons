============
Installation
============

*   How is the extension installed?

..  confval:: Using composer

    Example::

       composer require syradev/rt_pages_tree_icons




..  confval:: Using Extension Manager

    Where to download Extension zip?::

       https://extensions.typo3.org/extension/rt_pages_tree_icons



..  attention::
   Are there any dependencies that need to be resolved?

   This version requires TYPO3 V11LTS or TYPO3 V12LTS and PHP >= 8.0.0
